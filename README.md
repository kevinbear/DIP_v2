# DIP_v2
a simple digital image process homework program
使用平台:Linux

# 使用方法
1.創見資料夾
 Crtl+Alt+T
 $ mkdir /home/"username"/Desktop/DIP_v2
 $ cd /home/"username"/Desktop/DIP_v2
 $ mkdir /src
 
2.將資料放進資料夾
  "main.cpp"、"bmp.cpp"、"bmp.h"放入 src
  "*.bmp"、"Makefile" 放入 DIP_v2

3.終端機編譯程式
  $ make all

# 清除方式
  $ make clean
